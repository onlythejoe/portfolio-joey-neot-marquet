Laboratoire vivant : chaque prototype teste une architecture, une interaction ou une esthétique précise. Objectif : prouver par l’exemple que l’on peut faire mieux avec moins.

Chaque projet est une brique d’un système plus vaste. J’itère rapidement, j’intègre les leçons, je capitalise dans le framework pour accélérer les projets suivants.

Ce site est un journal de recherche visuel : les patterns testés ici nourrissent ensuite des produits réels (e-commerce, SaaS, expériences immersives).
