**Roadmap**
- Court terme (0-2 semaines)
  - Corriger assets manquants/doublons CSS, sécuriser a11y basique, mettre en place h1/page.
  - Unifier layout de base et clarifier l’ordre des imports (tokens > core > layout > patterns).

- Moyen terme (2-6 semaines)
  - Fusionner Design System : tokens consolidés, composants UX uniques, suppression EA legacy.
  - Refonte contenus prioritaires (Home, Portfolio, Contact) avec preuves, médias et CTA ciblés.
  - Router/loader v2 avec cache, aborts, erreurs explicites, SEO enrichi.
  - Responsive complet et motion system (timings + préférences utilisateurs).

- Long terme (6-12 semaines)
  - Pipeline de build (Vite/Rollup) + optimisation assets (font loading, bundling, minification, cache busting).
  - Bibliothèque de patterns “EA UX v2” documentée (story-like page ou docs internes).
  - Monitoring perf (Lighthouse CI) et QA (tests navigation, accessibilité, régressions CSS).
  - Exploration DA : set de visuels/galleries réels pour slots, animations premium GPU-friendly, gradients/textures signature.
