## SEO & Metadata
- Meta description dynamique via `seo.js` (fallback générique).
- OG/Twitter : titres, descriptions et site_name injectés; Twitter card summary par défaut.
- Canonical : présent sur `index.html`.
- JSON-LD : ItemList des pages principales ajouté dans `index.html`.

## Recos futures
- Ajouter `og:image`/`twitter:image` quand un visuel sera disponible.
- Générer des descriptions spécifiques par page (ajout dans pages.json) pour enrichir SEO.
