# Préviews — UX Enhancements

- Pattern commun : chaque preview enveloppée dans `.ux-stack`, ajout de `.ux-tags` (3–4 mots-clés) pour les signaux rapides, usage de `ux-lead` + `ux-paragraph` pour la hiérarchie, copy réduite à 2–3 lignes.
- home (hero, index, intro) : tags (Design/Systèmes/Code, Lab/Framework/UX, Design/Code/Pilotage), leads raccourcies, paragraphe mobile-friendly.
- about (index, bio, skills, vision) : tags portrait/hybridation/art/code/systèmes ; leads condensées ; suppression des pavés longs.
- expertises (index, DA, tech, conseils) : tags métier (DA/Webdesign/Motion, Frameworks/Automations/R&D, Vision/Roadmap/QA), leads précises, paragraphes uniques.
- portfolio (index, projects 001/002/003) : tags par projet, leads synthétiques, paragraphe succinct pour hook.
- vision/manifeste : tags systèmes/interfaces sensibles/intelligence créative, lead + paragraphe court.
